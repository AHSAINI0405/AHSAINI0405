### 📌 Featured Projects

#### 🔹 [Job Portal](https://github.com/AHSAINI0405)
Full-stack job portal supporting two distinct roles — Employer and Job Seeker — with JWT-based auth and role-based access control. 15+ REST endpoints, optimized MongoDB schemas, resume upload via Multer, email verification with Nodemailer, and real-time application tracking. Deployed on Render with an automated CI/CD pipeline.
`Node.js` `Express.js` `MongoDB` `JWT` `REST APIs`

#### 🔹 [Billify – Invoice SaaS Platform](https://github.com/AHSAINI0405)
Multi-tenant SaaS invoicing platform built for 100+ concurrent users, cutting invoice generation time by 60%. Secure auth via NextAuth, OAuth 2.0, and JWT with refresh-token rotation. GST-compliant invoice generation with PDF export, backed by Redis caching for faster dashboard loads.
`Next.js` `React.js` `TypeScript` `MongoDB` `Redis`

#### 🔹 [ResolveX – Civic Problem Management System](https://github.com/AHSAINI0405)
Real-time civic complaint management platform for simulated municipal departments, with Socket.io live status updates and Gemini AI-driven complaint classification. Microservices-inspired backend with auth, complaints, and notification services communicating via Redis pub/sub — cutting inter-service latency by 30%.
`React.js` `Node.js` `Express.js` `MongoDB` `Socket.io`
